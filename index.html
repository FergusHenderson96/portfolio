About me

Hi my name is Fergus and I'm a Full Stack Developer

I have recently completed Code Nation's 12 week developer bootcamp and I am very keen to further my skills in the industry. During the course I have learned basic web development skills using HTML, CSS and Javascript. These skills were useful when going on to create more advanced applications using React for UI as well as Node.js to create full stack applications. I linked these applications to Mongo DB in order to store data in the database such as login details and passwords. I have the ability to deploy applications using Heroku and Netlify so they can be hosted and accessed on the web. I use version control on all projects, storing code in repositories on Github and also collaborating with other developers using branches to merge work. Working on projects in teams has highlighted the importance of clear and concise code as well as adding precise notes to describe the functionality of the code to ensure other developers are able to understand my work. I have also learned the basics of C++ and creating video games in Unreal Engine in my spare time since completing the course whilst continuing to work on some cool projects including a munro site and meal prep app.
